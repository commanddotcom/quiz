# Веб-додаток для тестування

Простий і зручний додаток для проходження тестів на Vue.js та Tailwind CSS.

## Функціонал

- 📝 Вибір тесту зі списку доступних тестів
- 🎯 Проходження питань по черзі
- 📊 Відображення прогресу (номер поточного питання / загальна кількість)
- ✅ Підсвічування правильних відповідей зеленим
- ❌ Підсвічування неправильних відповідей червоним
- 🔄 Обов'язковий перезапуск при помилці
- 🖼️ Підтримка зображень у питаннях
- 👤 Введення імені після успішного проходження
- 🎉 Привітальне вікно з результатами

## Як запустити

### Варіант 1: Docker (рекомендовано) 🐳

```bash
# Запустити контейнер
docker-compose up -d

# Відкрити у браузері
http://localhost:8080
```

Зупинити:
```bash
docker-compose down
```

### Варіант 2: Без Docker

1. Відкрийте файл `index.html` у браузері
2. Виберіть тест зі списку
3. Відповідайте на питання
4. При помилці - починайте спочатку
5. Пройдіть всі питання правильно
6. Введіть своє ім'я
7. Зробіть скріншот фінального екрану!

## Структура JSON файлу

```json
{
    "tests": [
        {
            "id": 1,
            "name": "Назва тесту",
            "questions": [
                {
                    "id": 1,
                    "text": "Текст питання",
                    "image": "url або null",
                    "answers": ["Варіант 1", "Варіант 2", "Варіант 3"],
                    "correctAnswer": 0
                }
            ]
        }
    ]
}
```

## Додавання власних тестів

Відредагуйте файл `tests.json` та додайте свої тести за вказаною структурою.

- `id` - унікальний номер тесту
- `name` - назва тесту
- `questions` - масив питань
- `text` - текст питання
- `image` - URL зображення (або `null` якщо немає)
- `answers` - масив варіантів відповідей
- `correctAnswer` - індекс правильної відповіді (починаючи з 0)

## Технології

- Vue.js 3
- Tailwind CSS
- Vanilla JavaScript
